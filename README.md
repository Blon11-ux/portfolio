# portfolio
 Node.js & EJS Personal Portfolio

A professional, server-side rendered (SSR) portfolio website built to showcase my web development journey, technical projects, and skills.

 Live Demo
デプロイ済みの実機環境はこちらのリンクからご確認いただけます。: [https://memoapp-8eb2.onrender.com](https://memoapp-8eb2.onrender.com)

# Tech Stack
Runtime: Node.js
Framework: Express.js
Template Engine: EJS (Embedded JavaScript)
Styling: CSS3 (Flexbox & Grid)
Deployment:Render (CI/CD)

 Project Structure
├── views/              # EJS templates
│   ├── partials/       # Navbar & Footer
│   └── index.ejs       # Main Portfolio page
├── public/             # Static files (CSS, Images)
├── app.js              # Express server setup
└── package.json        # Dependencies



bash
# 1. Clone the repository
git clone https://github.com/Blon11-ux/portfolio.git

# 2. Navigate into the project folder
cd portfolio

# 3. Install dependencies
npm install

# 4. Start the development server
node app.js
